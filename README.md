# Legends
### Mobile app to search "League of Legends" champions

![App](https://github.com/cyberkaidev/Flutter-App---League-of-Legends/blob/master/readme/legends_gif.gif)

## Features
- Champions list view
- Favorite champions
- Champion Details

## Tech
- [Flutter] - App Development Toolkit!

## Installation
Install is very easy

```sh
flutter pub get
```
Open your emulator
```sh
flutter run
```
ready :)
